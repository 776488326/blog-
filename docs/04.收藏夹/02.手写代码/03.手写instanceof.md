## 手写instanceof

```js
// 沿着隐式原型链不断查找

  function instanceOf(instance,prototype){
      let i = instance;
      while(i){
          if(i.__proto__==prototype){
              return true;
          }
          i = i.__proto__;
      }
      return false;
  }
```
