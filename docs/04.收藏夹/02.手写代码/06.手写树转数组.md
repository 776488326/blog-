## 手写树转数组

```js
    // 递归
    function treeToArray(arr){
        return arr.reduce((prev,cur)=>{
            const {id,val,pid,children=[]} = cur;
            return prev.concat([{id,val,pid}],treeToArray(children));
        },[])
    }
```
