# 初识Vue3

## Vue3新特性

- 新的RFC机制允许所有人参与新语法的讨论
- 新的构建工具Vite
- 组合式API
- 新的响应式实现
- 新的内置组件Fragment，Teleport，Suspense
- 自定义渲染器
- 源码使用TypeScript重构

## 注意点

- 不兼容IE11
- vue-router支持Vue3，也采用了组合式APi风格
- vue-cli4支持Vue3，除了Vite也可使用Vue-cli来创建工程
- Vuex4也支持Vue3，除了Pinia可以使用Vuex4来进行数据管理
