# 常见的http状态码
1. 100  ——> 信息响应
2. 200  ——> 成功响应
    > 201：新创建了一个资源  
    > 204：无需返回内容
3. 300  ——> 重定向
    > 301/308：永久重定向  
    > 302/307：临时重定向
4. 400  ——> 客户端错误
    > 401：用户需要验证  
    > 403：拒绝执行  
    > 404：资源未找到  
    > 405：方法错误  
5. 500  ——> 服务端错误
    > 502：作为网关，上游服务器响应错误  
    > 503：作为网关，上游服务器响应超时
